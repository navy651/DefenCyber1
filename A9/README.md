# A9 – Privacy Techniques Used Online

## Overview
This project explores privacy techniques used online to protect user data, reduce tracking, and enhance anonymity. Modern web environments expose users to various tracking methods such as cookies, IP logging, and fingerprinting, making privacy protection increasingly important.

---

## Key Privacy Techniques

### 1. Virtual Private Networks (VPNs)
VPNs encrypt internet traffic and mask the user’s IP address, preventing tracking by websites, ISPs, and attackers.

### 2. Anonymous Browsing (Tor & Proxies)
- **Tor** uses onion routing to anonymize user traffic through multiple encrypted layers  
- **Proxies** hide IP addresses but provide weaker privacy due to lack of encryption  

### 3. Privacy-Focused Browsers
Modern browsers reduce tracking by:
- Blocking third-party cookies  
- Preventing fingerprinting  
- Limiting tracking scripts  

### 4. Encrypted Communication
End-to-end encryption ensures that only the sender and recipient can read messages, protecting sensitive data from interception.

### 5. Browser Privacy Tools
Privacy extensions and settings help:
- Block ads and trackers  
- Manage cookies  
- Prevent cross-site tracking  

---

## Real-World Example: Incognito Mode

I explored privacy techniques in practice using **Google Chrome Incognito Mode**.

Incognito mode prevents the browser from storing:
- Browsing history  
- Cookies and site data  
- Information entered in forms  

Additionally, it **blocks third-party cookies by default**, reducing tracking from advertisers and external websites.

However, incognito mode does not provide full anonymity, as activity may still be visible to:
- Websites visited  
- Internet Service Providers (ISPs)  
- Network administrators

## Tools Used

## Conclusion

Online privacy is achieved through a combination of tools, technologies, and user practices. Techniques such as VPNs, Tor, encryption, and browser privacy features work together to reduce tracking and protect personal data.

The use of incognito mode demonstrates how simple browser-based features can significantly enhance privacy by limiting local data storage and reducing tracking mechanisms.
