# A1 – Security Concepts at UWA (CIA Triad)

This folder contains my work for Activity A1, where I explored cybersecurity concepts implemented at The University of Western Australia based on the CIA Triad: Confidentiality, Integrity, and Availability.

## Included Evidence
- **screenshots/** – Photos and screen captures of UWA security-related systems, training, and policies.

## Summary
This activity investigates how UWA applies the CIA Triad through policies, training, incident response, and system reliability measures.
